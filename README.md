## Run the project locally
- Clone the project (https://github.com/dreamcatcher0101/homey-assessment.git)
- Run `bundle install`
- Run `rails db:create`
- Run `rails db:migrate`
- Run `rails db:seed`
- Run `rails s`
- Open your browser and go to `localhost:3000`
